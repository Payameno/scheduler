# Interview Scheduler

## Setup

Install dependencies with `npm install`.

## Running Webpack Development Server

```sh
npm start
```

## Running Jest Test Framework

```sh
npm test
```

## Running Storybook Visual Testbed

```sh
npm run storybook
```

## Dependencies
  axios : ^0.27.2
  classnames : ^2.2.6
  normalize.css : ^8.0.1
  react : ^16.9.0
  react-dom : ^16.9.0
  react-scripts : 3.4.4

  ## Scheduler API repo

  https://github.com/lighthouse-labs/scheduler-api
